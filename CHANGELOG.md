AngularBarebone
===============

**v0.1.0** [Feb. 22, 2014] Initial commit.

A project boilerplate using AngularJS library. boilerplate: angular
can run on Apache or NodeJS servers.

**Features:**

1. nodejs server
2. apache html5 mode
3. Grunt build
4. `.jshintrc` for your angularjs apps
